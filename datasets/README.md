# Datasets

## Description
The Rscript [`0_DataGeneration.R`](https://github.com/Jojo6297/edame/blob/main/datasets/0_DataGeneration.R) uses the package **curatedMetagenomicData** to obtain the microbial abundance datasets for 3 health states: Healthy [1-4], CRC (Colorectal cancer) [5-8] and IBD (Inflammatory Bowel disease) [9-13].
